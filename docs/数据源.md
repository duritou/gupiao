## mootdx (通达信 TCP 7709)

**职责：** 实时行情 + K线 + 五档盘口 + 逐笔成交

**优势：**
- TCP 二进制协议，不封 IP
- 延迟 ~50ms，比 HTTP 快 100 倍
- 46 字段返回（price, open, high, low, vol, amount, bid1~5, ask1~5）

**不支持：** PE, PB, ROE, 市值, 换手率（走腾讯财经或 tushare）

**已知 Bug：**
1. **frequency ≠ category** — `bars()` 参数名是 `frequency`，传 `category` 会被 `**kwargs` 静默吞掉，永远退化成日线
2. **BESTIP 空串 bug (0.11.x)** — 全新安装后 `Quotes.factory(market='std')` 可能抛 `ValueError: not enough values to unpack`。根因：`~/.mootdx/config.json` 的 `BESTIP.HQ` 初始是空字符串。**解决：显式传 server IP，不走 bestip**
3. **服务器会老化** — 官方 IP 列表不是永久有效的，需要维护备选服务器池
4. **海外 IP 不可用** — 通达信 TCP 协议走国内机房，海外环境全部超时。`tdx_client()` 会快速失败而非死等

**服务器池：**
```
119.97.185.59:7709 → 124.70.133.119:7709 → 116.205.183.150:7709
123.60.73.44:7709 → 116.205.163.254:7709 → 121.36.225.169:7709
123.60.70.228:7709 → 124.71.9.153:7709 → 110.41.147.114:7709
```

**频率映射表（mootdx 0.11.7 实测）：**
```
0=5分钟 1=15分钟 2=30分钟 3=60分钟 4=日线 5=周线 6=月线
8=1分钟 9=日线(默认) 10=季线 11=年线
```

**集成状态：** ✅ 已集成 (SourceManager, trust=动态)

## BaoStock

**职责：** A 股日线 K 线兜底

**优势：**
- 免费，无需认证
- 历史数据稳定，不会突然改版
- 支持复权

**限制：**
- 没有实时行情（只有日线收盘价）
- 速度较慢（~800ms/次）
- 不支持分钟线、财务数据

**代码格式：** `sz.000725`（不是 `000725.SZ`）

**集成状态：** ✅ 已集成 (SourceManager fallback, trust=动态)

## Tushare

**职责：** A 股财务数据（PE/PB/ROE/利润表/资产负债表/现金流量表）

**优势：**
- 财务数据最完整
- 有官方 SDK

**免费 Token 限制：**
- `daily` 接口需要积分（120 分+），免费 token 无权限
- `daily_basic` 接口免费，提供 PE/PB/市值/换手率（T+1）
- `stock_basic` 接口免费，提供股票基本信息
- `fina_indicator` 需要积分

**当前问题：**
- Token `c92945b...` 免费权限不足，无法调用 `pro.daily()`
- 已改用 `pro.daily_basic()` 获取 PE/PB/市值（T+1 数据）

**集成状态：** ✅ 已集成（仅 daily_basic + stock_basic）

## 东方财富 (EastMoney) — 独有数据

**只能从东方财富获取的数据（其他源没有）：**
- 龙虎榜席位 + 全市场龙虎榜
- 限售解禁日历
- 融资融券明细
- 大宗交易
- 股东户数变化
- 分红送转
- 个股资金流向（分钟/日级）
- 行业板块排名
- 研报列表/PDF 下载
- 个股新闻

**防封铁律：**
1. 串行，不并发 — 绝对不对东财开多线程
2. QPS ≤ 2 — 每次间隔 ≥ 1s + 随机抖动
3. 复用 HTTP 会话（Keep-Alive）
4. UA + Referer 必须正常

**集成状态：** ⏳ 待集成（仅用于独有数据，mootdx 能做的不用东财）

## 腾讯财经

**职责：** PE/PB/市值/换手率/涨跌停价/指数/ETF

**优势：**
- HTTP GET，不封 IP
- 88 字段 `~` 分隔
- 支持指数代码（000001=上证, 000300=沪深300, 399006=创业板）
- 支持 ETF 代码

**格式转换：** 6/9 开头 → `sh`，8 开头 → `bj`，其余 → `sz`

**集成状态：** ⏳ 待集成（mootdx 不提供 PE/PB/市值，需要腾讯补充）
