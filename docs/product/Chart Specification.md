# Chart Specification — AI Research Terminal

> Research 页面的图表规范和必须包含的图表类型。

---

## Charting Principles

1. **TradingView 风格** — 深色背景，清晰网格，专业配色
2. **交互完整** — 十字光标、缩放、拖拽、悬停提示
3. **指标可叠加** — 用户可在 K 线上叠加/取消 MA、EMA、BOLL 等
4. **周期可切换** — 分时、日K、周K、月K、半年、一年、五年
5. **性能优先** — Canvas 渲染，非 SVG，支持 5000+ 根 K 线

---

## Chart 1: K-Line (Candlestick) — 核心图表

**位置**: Research 页面主区域，占据约 70% 面积

**必须包含**:
```
┌─────────────────────────────────────────┐
│  [分时] [1日] [1周] [1月] [半年] [1年]  │  ← 周期切换
│  [MA5] [MA10] [MA20] [MA60] [EMA]     │  ← 指标叠加
│  [BOLL] [SAR]  [筹码]                    │
│                                         │
│      ┃                                  │
│      ┃    ┏━┓                           │
│      ┃   ┃   ┃     ┏━┓                 │
│      ┃  ┃     ┃   ┃   ┃   ┏━┓         │
│      ┃ ┃      ┃  ┃    ┃ ┃   ┃         │
│      ┃┃       ┃┃      ┃┃    ┃         │
│      ┃        ┃        ┃     ┃         │
│      ┗━━━━━━━━┛        ┗━━━━━┛         │
│                                         │
├─────────────────────────────────────────┤
│  成交量                                  │
│  ██ ██ ████ ██ ██████ ██ ████          │
└─────────────────────────────────────────┘
```

**蜡烛图颜色**:
- 阳线 (涨): `--up` (#22C55E) — 实体填充
- 阴线 (跌): `--down` (#EF4444) — 实体填充
- 空心/实心: 由数据决定（当前价 vs 前收盘价）

**MA 颜色**:
- MA5: `#F59E0B` (黄)
- MA10: `#3B82F6` (蓝)
- MA20: `#A78BFA` (紫)
- MA60: `#F97316` (橙)
- EMA: `#EC4899` (粉)

**交互**:
- 鼠标滚轮: 缩放
- 拖拽: 平移
- 十字光标: 显示 OHLC + 日期
- 双击: 重置视图
- 右键: 上下文菜单 (添加指标/切换周期/截图)

## Chart 8: Radar Chart (雷达图)

**位置**: 右侧 AI 面板

**维度** (5 轴):
- 技术面 (Technical)
- 基本面 (Fundamental)
- 资金面 (Capital Flow)
- 行业面 (Industry)
- 情绪面 (Sentiment)

**规格**:
- 尺寸: 160×160px
- 填充: `--ai-primary` 20% 透明度
- 边框: `--ai-primary`
- 刻度: 0-100，5 圈
- 无交互，纯展示

## Chart 10: Score Timeline (评分趋势)

**位置**: Timeline 页面

**规格**:
- 折线图，30 天
- X 轴: 日期 (MM-DD)
- Y 轴: 评分 (0-100)
- 折线颜色: `--ai-primary`
- 数据点: 4px 圆点
- 悬停: 显示日期 + 分数 + 变化原因

## Chart Tools (通用)

**十字光标 (Crosshair)**:
- 跟随鼠标移动
- 竖线: 贯穿所有指标区
- 横线: 仅在当前指标区
- 右上角浮动提示框: OHLC + 日期
- 按 `Alt` 锁定十字光标

**缩放 (Zoom)**:
- 鼠标滚轮: 水平缩放
- `+` / `-`: 键盘缩放
- 双击: 重置

**平移 (Pan)**:
- 拖拽: 水平平移
- `←` / `→`: 键盘平移

**标记 (Annotation)**:
- 金叉/死叉自动标记
- MA 突破自动标记
- 用户可手动添加标记 (v2.1)
