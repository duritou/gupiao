# User Flow — AI Research Terminal

## Daily Workflow

> 用户的一天：从打开终端到收盘复盘。

---

## 09:00 — Morning Brief (自动)

```
打开软件
    │
    ▼
┌─────────────────────────────┐
│  ☀ Daily Brief              │
│                             │
│  市场情绪: ★★★★☆ 72分       │
│  上涨 3,865  成交 1.43万亿   │
│  热点: AI · 半导体 · 机器人  │
│  Top 3: 寒武纪 91 · 中芯 86 │
│  风险: 新能源走弱            │
│  一句话: ...                 │
└─────────────────────────────┘
```

**用户动作**: 浏览（30s）→ 了解今天市场全貌 → 进入 Dashboard

---

## 09:05 — Dashboard 扫描

```
Dashboard
    │
    ├─ 看指数（上证/深证/创业板/科创50）
    ├─ 看热点板块（哪些在涨？）
    ├─ 看风险（有没有踩雷？）
    ├─ 看今日机会 Top 8
    └─ 看自选股评分变化
```

**用户动作**: 
- 发现半导体强势 → 点进去看
- 看到自选股某只评分大幅变化 → 点进去研究
- 发现风险预警 → 确认是否需要减仓

---

## 09:10 — Watchlist 监控

```
Watchlist (30s 自动刷新)
    │
    ├─ 扫描自选股评分
    ├─ 关注趋势变化（↑ / ↓）
    ├─ 发现新信号（MACD金叉 / 放量突破）
    └─ 点击进入深度研究
```

**用户动作**: 持续监控，发现信号 → 进入 Research 页面

---

## 09:15 — Research 深度研究

```
Research Page
    │
    ├─ 顶部: 股票名称、价格、涨跌幅、基本信息
    ├─ Tab: AI分析 | 分时 | 日K | 周K | 月K | 资金 | 财务 | 新闻
    │
    ├─ 主区域 (70%): K线图
    │   ├─ 蜡烛图 + MA5/10/20/60
    │   ├─ 成交量
    │   ├─ MACD
    │   ├─ RSI
    │   └─ KDJ
    │
    └─ 右侧面板 (30%): AI 分析
        ├─ AI Score + 星级
        ├─ 推荐 (强烈买入→卖出)
        ├─ 雷达图 (技术/基本/资金/行业/情绪)
        ├─ Evidence Cards
        ├─ 风险提示
        ├─ 关键指标
        └─ 资金流向
```

**用户动作**:
- 看 K 线（技术分析）
- 看 AI Score + Evidence（AI 怎么说？）
- 看资金流向（主力在干嘛？）
- 看财务（基本面怎么样？）
- 决策: 加入 Watchlist / 设置 Alert / 标记关注

---

## 10:00 — Compare 对比

```
Compare Page
    │
    ├─ 选择股票 A: 贵州茅台
    ├─ 选择股票 B: 五粮液
    └─ 并排对比:
        ├─ AI Score: 92 vs 87
        ├─ MACD: ✓ vs ✗
        ├─ RSI: 68 vs 61
        ├─ 估值: 偏高 vs 合理
        └─ AI建议: 持有 vs 可买
```

**用户动作**: 多个候选标的之间做选择

---

## 10:30-14:30 — 盘中监控 + Alert

```
Alert Center (30s 自动刷新)
    │
    ├─ 09:35  寒武纪  MACD金叉  91 ▲
    ├─ 10:14  北方华创  放量突破  87 ▲
    ├─ 11:05  茅台  北向加仓  78 ▲
    ├─ 13:15  招商银行  跌破MA20  42 ▼
    └─ 14:23  浪潮  MACD死叉  38 ▼
```

**用户动作**:
- 盘中持续接收 Alert
- Alert 出现 → 点击跳转 Research
- 判断是否需要操作

---

## 15:00 — 收盘

```
收盘后:
    │
    ├─ 查看 Timeline（今天评分变化？）
    ├─ 对比自己的判断 vs AI 评分
    └─ 等待 Daily Brief
```

---

## 18:00 — Daily Brief (收盘复盘)

```
☀ Daily Brief (收盘版)
    │
    ├─ 今日市场总结
    ├─ 我的自选股表现
    ├─ AI 预测回顾（7天前预测 vs 实际）
    ├─ 今日预警回顾
    └─ 明日关注
```

**用户动作**: 复盘今日，规划明日

---

## Weekly Flow

```
周一早上: 看周报 → 制定本周策略
周中: 日常监控 + 研究
周五收盘: 看本周收益 → 回溯 AI 预测准确率
```

---

## Flow Principles

1. **零摩擦启动**: 打开软件 → Dashboard 自动加载，无需任何点击
2. **渐进式深入**: 概览 → 列表 → 详情，每一步都是上一步的自然延伸
3. **被动接收 + 主动探索**: Alert 是被动的，Research 是主动的
4. **可中断可恢复**: 随时可以离开，回来时状态不变
5. **AI 无处不在但不出跳**: 每页都有 AI 辅助，但不打断用户
