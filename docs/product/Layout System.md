## Zone Definitions

### Top Bar (48px)

```
┌──────────────────────────────────────────────────────────┐
│ 🔍 [搜索股票/代码...]    │ 上证 3350 +0.85% │ 深证 +1.12% │ │
│                          │ 创业板 +1.45%    │ 科创 +0.92% │ │
└──────────────────────────────────────────────────────────┘
```

- 高度: 48px
- 背景: `--bg-panel`
- 底部边框: 1px `--border-default`
- 内容: 搜索框(左) + 指数条(右)
- 始终可见，不随页面滚动

### Left Nav (220px)

```
┌─────────────────┐
│                 │
│  AI Research    │  ← Logo 区域 (48px)
│  Terminal       │
│                 │
├─────────────────┤

│  🗺 Market Map  │
│  🔬 Research    │  ← 导航项 (40px 每项)

│  📚 Knowledge   │
├─────────────────┤
│  ⚙ Settings     │  ← 底部固定
└─────────────────┘
```

- 宽度: 220px
- 背景: `--bg-panel`
- 右边框: 1px `--border-default`
- Logo 区: 48px 高
- 导航项: 40px 高，图标 20px + 文字 14px
- Active 状态: 左边框 3px `--ai-primary`，背景 `--bg-hover`
- 始终可见

### Main Content (flex fill)

- 背景: `--bg-root`
- 内边距: 24px
- 纵向滚动
- 内部按页面布局自由划分
