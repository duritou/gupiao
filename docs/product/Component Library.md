# Component Library — AI Research Terminal

> 所有页面共享的 UI 组件。每个组件有固定结构和样式，不允许页面自行修改。

---

## 1. Score Badge

**用途**: 展示 AI 评分 (0-100)

```
┌──────────┐
│    92    │  ← --text-3xl (36px) --font-bold
│  ★★★★★  │  ← --text-sm, --warning
│   BUY    │  ← --tag-buy
└──────────┘
```

**变体**:
- `score >= 80`: 绿色 · ★★★★★ · Strong Buy
- `score >= 65`: 蓝色 · ★★★★☆ · Buy
- `score >= 50`: 灰色 · ★★★☆☆ · Neutral
- `score >= 35`: 黄色 · ★★☆☆☆ · Watch
- `score < 35`: 红色 · ★☆☆☆☆ · Sell

**规格**:
- 宽度: 120px
- 高度: auto
- 背景: `--bg-panel`
- 边框: 1px `--border-default`
- 圆角: `--radius-lg` (8px)

## 2. Evidence Card

**用途**: AI 分析证据链

```
┌──────────────────────────────────────┐
│ ✓ MACD形成金叉              可信度 95% │  ← 左图标 + 右百分比
│   DIF上穿DEA，柱状图由绿转红           │  ← 描述 (--text-sm, --text-secondary)
│   来源: MACD Signal    贡献: +12分     │  ← 来源 (--text-xs, --text-muted)
│   ████████████████████████░░░░  95%  │  ← 可信度条
└──────────────────────────────────────┘
```

**变体**:
- `icon: "check"` → 绿色左边框，✓ 图标
- `icon: "warning"` → 黄色左边框，⚠ 图标
- `icon: "info"` → 蓝色左边框，ℹ 图标

**规格**:
- 内边距: 12px 16px
- 左边框: 3px solid
- 底部间距: 8px
- 可信度条: 4px 高，`--bg-input` 底色，填充色跟随左边框


## 13. Search Input

**用途**: 股票搜索

```
┌─────────────────────────────────────┐
│  🔍 搜索股票代码或名称...            │
└─────────────────────────────────────┘
```

**行为**:
- 聚焦: 边框 `--border-active`
- 输入: 下拉显示匹配结果 (最多 5 条)
- 选中: 跳转 Research 页面
- 快捷键: `/` 聚焦搜索框
