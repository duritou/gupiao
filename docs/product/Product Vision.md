# Product Vision — AI Research Terminal v2.0

## One-Line Definition

> **AI Research Terminal 是一款 AI 驱动的专业股票研究终端。**

## What It IS

- **研究终端 (Research Terminal)** — 用户研究股票、行业、市场的工具
- **AI 增强 (AI-Augmented)** — AI 解释数据，不做决策
- **Daily Driver** — 用户每天早上打开的第一款软件
- **Professional** — 对标 Bloomberg Terminal、TradingView、同花顺 PC 客户端

## What It IS NOT

- ❌ 聊天机器人 — AI 在右侧面板解释，不是对话
- ❌ 量化交易平台 — 不做自动下单、程序化交易
- ❌ 后台管理系统 — 不是 Admin Dashboard 风格
- ❌ 移动端 App — PC 优先（1920×1080），移动端是 v2.1+
- ❌ 社交平台 — 不做股吧、评论、社区

## Reference Products

| 产品 | 借鉴什么 |
|------|---------|
| **Bloomberg Terminal** | 高信息密度、键盘快捷键、专业感 |
| **TradingView** | K线交互、图表质量、指标叠加 |
| **同花顺 Level2** | 市场深度、板块热力图、自选股体验 |
| **东方财富 PC** | 资金流向、新闻整合、F10 资料 |
| **Wind Terminal** | 数据深度、研报整合、行业分类 |

## Core Differentiator

```
传统股票软件 = 数据 + 图表
AI Research Terminal = 数据 + 图表 + AI 解释

AI 的角色：
  ✓ 解释 K 线形态
  ✓ 分析多指标共振
  ✓ 关联行业知识库
  ✓ 生成证据链
  ✓ 追踪预测结果
  ✓ 输出结构化报告

AI 不做的：
  ✗ 不代替用户做买卖决策
  ✗ 不给"必涨"承诺
  ✗ 不隐藏推理过程
```

## User Persona

**主要用户**: 有经验的 A 股投资者，需要专业工具辅助研究决策

- 每天看盘 2-6 小时
- 需要快速扫描市场、发现机会
- 需要深度研究单个标的
- 需要 AI 辅助解释但不替代判断
- 需要追踪自己的预测准确率

## Success Metric

> 用户每天早上打开 AI Research Terminal → 看 Dashboard → 扫 Watchlist → 研究标的 → 收到 Alert → 晚上看 Daily Brief → 第二天继续。

**不是**: DAU、留存率。  
**而是**: 用户是否形成了这个 Daily Workflow。
